***************************************************
НАЗНАЧЕНИЕ УТИЛИТЫ:

Утилита парсит XLSX-файл с данными и выдает на выходе CSV-файлы с именами вида "ИмяМашины_Shift_НомерСмены_Report.csv". Файл содержит таблицу вида:

Date,Scrap Recycle,Scrap
2020-03-17,24,5
.....





***************************************************
НАСТРОЙКИ УТИЛИТЫ:

Настройки хранятся в двух файлах в директории программы: Scrap_Settings.csv и config.cfg



*** Файл Scrap_Settings.csv ***
Рекомендуется редактировать в Excel. Файл содержит информацию о столбцах таблицы, в которых находятся исходные данные для обработки. Файл имеет внутри следующую структуру:

Machine Name;Column 1;Column 2
Tread1;EU;FA
Tread2;EV;FB
.....

Первая строка - заголовок таблицы
Далее файл заполняется строками, в которых ищутся данные. Первым в строке идет имя машины, далее следует имя первого столбца таблицы рабочей книги, откуда берутся данные Scrap Recycle, вторым указывается имя столбца таблицы, откуда берутся данные Scrap.

Эти данные затем будут помещены в CSV-файлы, генерируемые утилитой.



*** Файл config.cfg ***

Файл можно редактировать в любом текстовом редакторе. Перед знаком = и после не должно быть пробелов! Знак # начинает комментарий. Файл содержит настройки, указывающие на местоположение рабочей книги, которая будет обрабатываться утилитой, директорию для вывода и т.п. В файле уже есть комментарии по каждому из параметров.
